# AmIHot 1.0.0


This is a program to monitor the temperature of the PC and display it to the bar charts in the terminal or in text mode

# Programs and libraries need to compile and work
 - cmake
 - lm_sensors
 - ncurses
 - libsensors
 
# How to install:
<b>./configure<br/>
 make <br/>
 sudo make install </b>
